Carousels

The SonyCarousel class creates a number of carousel variations. "sony-carousel" classes (see Jade example below) provide basic styles, and are used throughout this page, but you may create your own implementation of the CSS so long as:

  1. The carousel wrapper (.sony-carousel-wrapper in examples) is overflow: hidden; and is position: relative/absolute;
  2. The carousel slider (.sony-carousel in examples) contains all the slides (.sony-carousel-slides in examples) from left to right, whether by absolute/relative positioning or floats. (Though should at least be position: relative;)
  3. If you're making a looped carousel, your CSS should allow for items to be dynamically added on both ends of the carousel without breaking the layout.

More detailed documentation of the Javascript class is located here.

Example (Jade)

  div#carousel-(number)
    div.sony-carousel-wrapper
      div.sony-carousel
        div.sony-carousel-slide
          div.sony-carousel-slide-children
          div.sony-carousel-slide-children
        div.sony-carousel-slide
          div.sony-carousel-slide-children
          div.sony-carousel-slide-children
        div.sony-carousel-slide
          div.sony-carousel-slide-children
          div.sony-carousel-slide-children

Overflow-visible, grid-based, flexible carousels (OSC-style)

This is an example of a carousel that, while the slides are floated, also lines the active slide up with the grid. With the `.sony-carousel` class, slides with the `.sony-carousel-slide` class that *also* have a class of `.slimgrid` will inherit a common right margin, creating the desired effect. Slide children can therefore have the usual span and offsets as broken down in the scaffolding section of documentation.

Slide A2

Slide B2

Jade
  div.carousel-7
    div.sony-carousel-wrapper
      div.sony-carousel
        div.sony-carousel-slide.slimgrid
          div.sony-carousel-slide-children.t1.span6 Slide A1
          div.sony-carousel-slide-children.t1.span6 Slide A2
        div.sony-carousel-slide.slimgrid
          div.sony-carousel-slide-children.t1.span6 Slide B1
          div.sony-carousel-slide-children.t1.span6 Slide B2
        div.sony-carousel-slide.slimgrid
          div.sony-carousel-slide-children.t1.span6 Slide C1
Javascript
  $('.carousel-7 .sony-carousel').sonyCarousel({
    wrapper: '.sony-carousel-wrapper',
    slides: '.sony-carousel-slide',
    slideChildren: '.sony-carousel-slide-children',
